
# Lead Scrapper

A simple scrapper that scraps lead data from google maps


## Run Locally

Clone the project

```bash
  git clone https://github.com/S-jenishPatel/LeadScrapper.git
```

Go to the project directory

```bash
  cd LeadScrapper
```

Initialise project

```bash
  uv init
```

Install dependencies

```bash
  uv sync
```

Install playwright chromium browser binaries

```bash
  playwright install chromium
```

Start the script

```bash
  uv run python app/main.py
```
